Amazon-eBay API Smashup
Welcome to my Amazon-eBay API mixup! The concept here is pretty simple: pick a product category, a price range, a desired sales rank, and enter a search term. The application will return you results for each product on Amazon and eBay, which will help identify areas for price-arbitrage.
